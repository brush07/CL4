## 📌 Project Overview

This project demonstrates data visualization using **Tableau**.
The objective is to analyze sales data and present insights through interactive charts, dashboards, and storytelling.

---

## 🎯 Objectives

* Connect to dataset
* Build multiple visualizations
* Analyze sales performance
* Create an interactive dashboard
* Present insights using story points

---

## 🛠️ Tools Used

* Tableau (Cloud)
* CSV Dataset (Sales Data)

---

## 📊 Visualizations Created

* Sales by Territory (Bar Chart)
* Monthly Sales Trend (Line Chart)
* Product Performance (Category Analysis)

---

## 📋 Dashboard Features

* Interactive filtering using **“Use as Filter”**
* KPI-style insights
* Clean and structured layout
* User-friendly design

---

## 📖 Story Points

* Overview of sales performance
* Trend analysis over time
* Product-level insights

---

## 🔍 Key Insights

* Certain territories generate higher sales compared to others
* Sales trends vary across different time periods
* Some product lines contribute significantly to revenue
